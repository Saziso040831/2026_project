# 📦 Lost and Found Management System

## 📌 Project Overview
The Lost and Found Management System is a web-based application developed to help users report, search, and recover lost items within a university environment. The system improves efficiency in handling lost property by providing a centralized digital platform.

---

## 🎯 Objectives
- To allow users to report lost and found items
- To provide a searchable database of items
- To improve recovery of lost belongings
- To create a user-friendly web interface
- Organize lost items into categories 

---

## 🛠️ Technologies Used
- **HTML (83.1%)** – Structure and layout of the website  
- **Python (16.9%)** – Backend logic and data processing  

---

## ⚙️ Features
- 🔍 Search for lost items  
- ➕ Report lost items  
- 📦 Report found items  
- 👤 User interaction with the system  
- 📄 Display item details
